International Student Mental Health Analysis

Objective
To analyze how length of stay impacts mental health diagnostic scores 
(PHQ-9, SCS, ASISS) among international students.

SQL Approach
- Filtered for international students
- Grouped data by length of stay
- Calculated:
  - Count of students
  - Average PHQ score
  - Average SCS score
  - Average ASISS score

Tools Used
- SQL
- Student survey dataset
